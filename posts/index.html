如何用 hugo 搭建个人博客

Hugo是一个用Go编写的静态站点生成器，主题丰富，生成速度快。 安装HUGO 在HUGO官网下载Windows版本或者iOS版本 下载解压，把hugo.exe文件保存在一个目录里，比如D:/software/hugo 配置环境变量，把D:/software/hugo加到path 添加完后在终端运行hugo version显示版本说明安装成功 快速搭建博客 进入HUGO官网，点击Quick Start开始 会看到配置指南，依次复制step1~setp7的命令直到出现public目录，这个目录就是我们需要提交到GitHub上的文件 建立GitHub仓库 你需要在github上创建repository，名字就是<你的用户名>.github.io，创建完后，返回你本地命令行，进入public目录，执行以下命令： git init git add . git commit -m "Initial commit." git remote add origin git@github.com:Brent-Li/brent-li.github.io.git git push -u origin master 最后打开<你的用户名>.github.io网址，就可以看到你的个人博客。
